### Current Goal
Implement the end-to-end Python engine as a single structured package, designed to execute sequentially via a CLI script (`main.py`). The pipeline must run smoothly on a machine with an NVIDIA RTX 4050 Laptop GPU (6GB VRAM), ensuring strict INT8 quantization to avoid Out-of-Memory (OOM) exceptions.

### Architectural Requirements

1. File Architecture:
Create a modular project structure:
├── app/
│   ├── __init__.py
│   ├── transcription.py    # faster-whisper implementation
│   ├── segmentation.py     # Ollama text parsing & FFmpeg extraction
│   ├── reframing.py        # OpenCV + MediaPipe face tracking & crop coordinates
│   └── subtitles.py        # ASS file generation & hardcoding
└── main.py                 # CLI Orchestrator using Typer or Argparse

2. Technical Specifications per Module:

* transcription.py: 
  - Implement a class `LocalTranscriber`.
  - Instantiate `WhisperModel("large-v3-turbo", device="cuda", compute_type="int8")`.
  - Include a function to extract audio to a lightweight WAV first.
  - Return word-level timestamps (`word_timestamps=True`) inside a structured JSON/dictionary format.

* segmentation.py:
  - Connect to a local Ollama instance running `llama3` or `mistral`.
  - Use the exact system prompt from the blueprint. Handle potential JSON formatting errors from the LLM gracefully with a fallback parser.
  - Call FFmpeg using `subprocess` with the `-c copy` flag to instantaneously cut the 3 standalone stories based on the LLM's timestamps.

* reframing.py:
  - Implement face tracking using `MediaPipe Face Detection`.
  - Write a moving-average smoothing calculation using a sliding window of 12 frames to prevent jerky camera movements.
  - Calculate the 9:16 bounding box (`w:h:x:y`) relative to a 16:9 source frame.
  - Output an FFmpeg command sequence using `-vf crop=w:h:x:y` to crop the segments.

* subtitles.py:
  - Write a utility that groups words into max 2-3 words per line based on timestamps.
  - Dynamically generate an Advanced SubStation Alpha (.ass) file text block with styling (Font: Arial Black, Primary Color: White, Outline Color: Black, dynamic scaling).
  - Provide the FFmpeg command string to burn the ASS file into the video.

3. Orchestration (`main.py`):
  - Tie the pipeline together: Input Video -> Transcribe -> Get Sub-clips via Ollama -> Reframe each sub-clip to 9:16 -> Generate Kinetic Subtitles -> Burn Subtitles into final vertical output.
  - Print clean, structured terminal logs for each step showing processing times.

Please generate the complete, production-grade Python code for all the modules listed above, ensuring proper type hinting, clean docstrings, and robust error handling for subprocess calls.
